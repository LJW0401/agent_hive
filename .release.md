## 使用方式

先 `cd` 到二进制文件所在目录，后续命令都在该目录下执行。

### 运行

```bash
# 前台运行
./agent-hive run --config config.yaml
```

浏览器访问 `http://localhost:8090`。局域网其他设备访问 `http://你的IP:8090`。

### 作为系统服务运行

```bash
# 安装 systemd 服务（需要 sudo）
sudo ./agent-hive install --config /path/to/config.yaml

# 管理服务
sudo ./agent-hive start
sudo ./agent-hive stop
sudo ./agent-hive restart
./agent-hive status
./agent-hive logs -f

# 卸载服务
sudo ./agent-hive uninstall
```
